## [Markdown to PDF](https://github.com/simonhaenisch/md-to-pdf)
```shell
$ git clone git@github.com:kindaidai/Curriculum-Vitae.git
$ cd Curriculum-Vitae
$ yarn
$ md-to-pdf CURRICULUM-VITAE.md
```
